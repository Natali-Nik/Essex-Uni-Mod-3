Overview of Applied Learning Unit 7-10

Perceptrons, CNN functionality for object recognition and its ethical implications; preparation for neural network development using Python and CIFAR-10.

Key Outcomes and Lessons

By the units mentioned below, I gained a foundational knowledge of simple and multi-layer perceptrons, and Convolutional Neural Networks. The exercise on CNNs in the context of object recognition helped to prepare for the assignment in Unit 11, where we encountered a similar task. Unit 7: Perceptron Activities

These Python activities provided a foundation for understanding perceptrons, essential for developing Artificial Neural Networks (ANNs) and their applications. Tasks included basic perceptron implementation (simple_perceptron.ipynb), modeling the AND operator, and using multi-layer perceptrons with the sigmoid function for non-linear decision boundaries (multi-layer_perceptron.ipynb). These exercises improved my comprehension, understanding of Python coding structures, and of Machine Learning's applicability and challenges with different datasets.

Unit 9: Neural Networks for Object Recognition

In this exercise, the Convolutional Neural Network (CNN) model for object recognition was run and reviewed, with input images modified to observe and validate predictions. This hands-on exploration clarified CNN functionality, particularly for object recognition, and was a good preparation for developing a neural network in Unit 11 using Python and the CIFAR-10 dataset. The knowledge gained in Module 2 on Numerical Analysis, particularly on statistical methods and visualizations related to Validation, Accuracy, and Loss, became clearer and more meaningful when applied in practice.

Unit 9: Exploring the Ethics of CNNs in Modern AI

An article by Wall (2019) explores the advancements, uses, and ethical/regulatory concerns surrounding facial recognition technology, one of which is the risk of bias against certain demographics due to flawed training datasets. I learned that while this technology can enhance security, its deployment raises significant ethical questions and calls for stricter regulations to prevent misuse, especially in the context of government surveillance and authoritarian practices.
