Unit 7: Perceptron Activities

These Python activities provided a foundation for understanding perceptrons, essential for developing Artificial Neural Networks (ANNs) and their applications. Tasks included basic perceptron implementation (simple_perceptron.ipynb), modeling the AND operator (perceptron_AND_operator.ipynb), and using multi-layer perceptrons with the sigmoid function for non-linear decision boundaries (multi-layer_perceptron.ipynb). These exercises improved my comprehension, coding skills, and understanding of machine learning's applicability and challenges with different datasets.

Unit 9: Convolutional Neural Networks (CNN) for Object Recognition

In this exercise, the CNN model for object recognition was run and reviewed, with input images modified (e.g., plt.imshow(x_test[16]) to observe and validate predictions. This hands-on exploration clarified CNN functionality, particularly for object recognition, and provided valuable preparation for developing a neural network in Unit 11 using Python and the CIFAR-10 dataset.
