Overview of Applied Learning Unit 3-6

Exploring Diverse Applications: Data Analysis, Python Programming with Google Colab, and Jaccard Coefficient Theory in Practice.

Key Lessons and Outcomes

By the units mentioned below, I learned about how data variations impact correlations and regression models (Unit 3) while developing skills in Python and working efficiently with Google Colab. Additionally, in Unit 5, I gained an understanding of the Jaccard coefficient and Jaccard distance, their practical applications in comparing sets, and improved my problem-solving abilities through hands-on practice with calculations.

Unit 3: Correlation and Regression

In Unit 3’s e-Portfolio Activity we had to run four programs in Jupyter Notebook (via Google Colab) to explore how changes in data points affect correlations and regression models. Analyzed car database regression techniques including linear, multiple linear, polynomial regression, and covariance-Pearson correlation. This activity provided practical insights into the relationships between variables and the impact of data variations on analysis as well as first skills in efficiently working within Google Colab and Python programming language.

Unit 5: Jaccard Coefficient Calculations

In Unit 5’s e-Portfolio Activity Jaccard Coefficient Calculations, we calculated the Jaccard coefficient for provided data pairs to measure the similarity between two sets of labels with distinguishing parameters. This involved understanding how the coefficient, ranging from 0 (no similarity) to 1 (complete similarity), and its complement, the Jaccard distance, quantify similarity and dissimilarity. Beyond learning the concept, I also explored practical applications of this calculation. While initially challenging, having correct results for comparison proved useful in improving my understanding and ability in applying the method.
